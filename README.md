# cv-typst

A Typst CV template with a configurable design system.

## Installation

Add as a Git submodule in your project:

```sh
git submodule add https://github.com/tenyoru/cv-typst template
```

Then import in your `.typ` file:

```typst
#import "template/cv.typ": *
```
